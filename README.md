# درب التفوق — موقع الكتاب (حزمة Replit)

صفحة هبوط تسويقية لكتاب **درب التفوق** — عربية، اتجاه RTL، خط Cairo.
هذه حزمة **جاهزة للتشغيل** ومنظّمة في ملفات منفصلة قابلة للتحرير.

---

## 🚀 التشغيل في Replit

1. أنشئ Repl جديد من نوع **HTML, CSS, JS** (أو **Static / Node**).
2. ارفع كل محتويات هذا المجلد كما هي (مع الحفاظ على بنية المجلدات).
3. اضغط **Run** — سيُفتح `index.html` مباشرةً. لا حاجة لأي تثبيت (`npm install`).

> الموقع يعمل بالكامل في المتصفح (Static). لا يحتاج خادماً خلفياً.

---

## 📁 بنية الملفات

```
index.html              ← نقطة الدخول (افتح هذا)
site.css                ← تنسيقات الموقع
parts/                  ← أقسام الصفحة (React + JSX)
  ├─ App.jsx            ← يجمع كل الأقسام
  ├─ Shared.jsx         ← الهيدر/الفوتر + أدوات مشتركة
  ├─ Hero.jsx           ← القسم الرئيسي (Hero)
  ├─ Learn.jsx          ← ماذا ستتعلّم
  ├─ Chapters.jsx       ← الأقسام الخمسة
  ├─ Preview.jsx        ← معاينة صفحة داخلية
  ├─ Authors.jsx        ← المؤلفون
  ├─ Order.jsx          ← الطلب والسعر
  └─ Avatars.jsx        ← صور المؤلفين النائبة
assets/                 ← الصور والأيقونات
  ├─ icons.js           ← مجموعة الأيقونات (window.DRTFIcons)
  ├─ inline-images.js   ← الصور مضمّنة base64 (شعار/غلاف/صفحة)
  ├─ logo.png · cover-front.jpg · page-section1.png · ...
_ds/                    ← نظام التصميم «درب التفوق» (ألوان/خطوط/مكوّنات)
```

---

## 🎨 نظام التصميم

- الألوان والخطوط والمسافات معرّفة كـ **CSS variables** داخل `_ds/.../tokens/`.
- المكوّنات (Button, Banner, Card, SectionTag...) متاحة على `window.DesignSystem_9d24f7`.
- اللون الأساسي: زيتوني داكن `#3E402D` · الأخضر `#BBBF73` · تدرّج العلامة أصفر→أخضر→تركواز.
- الخط: **Cairo** (يُحمّل من Google Fonts).

---

## 🤖 لإعادة البناء عبر مساعد Replit

إن أردت أن يعيد مساعد Replit بناء الموقع بتقنية أنظف (مثل React/Vite أو Next.js)،
انسخ له هذا الطلب:

> «هذا موقع هبوط عربي (RTL) لكتاب اسمه (درب التفوق). أعد بناءه بنفس التصميم والمحتوى
> باستخدام React + Vite. حافظ على: الخط Cairo، الاتجاه RTL، الألوان الزيتونية/الخضراء،
> الأقسام بالترتيب (Hero، ماذا ستتعلّم، الأقسام الخمسة، معاينة، المؤلفون، الطلب)،
> وكل النصوص كما هي. الكود المرجعي في مجلد parts/ والتصميم في _ds/.»

---

## ✏️ تعديل سريع

- **النصوص:** افتح ملف القسم في `parts/` وعدّل النص مباشرةً.
- **الألوان:** غيّر متغيّرات `_ds/.../tokens/colors.css`.
- **الصور:** استبدل ملفات `assets/` بنفس الأسماء.
